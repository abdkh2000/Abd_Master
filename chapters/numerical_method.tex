\chapter{Numerical Method}
\label{ch:numerical-method}

This chapter will document the numerical formulation and the active BRINE code
path used for the project calculations. It is intentionally limited to a
structured placeholder until the relevant implementation details have been
traced and reconciled with the exact compiled configuration. No numerical
method is inferred here from filenames, inactive branches, or unexecuted input
settings.

\section{Solver Formulation}

The final version will describe the spatial discretisation, temporal
integration, pressure treatment, scalar coupling, and boundary implementation
that are confirmed in the active source. Equations already established in
Chapter~\ref{sec:theoretical-background} will be referenced rather than
repeated.

\section{Parallel Decomposition and Accelerator Execution}

The final version will document the source-confirmed domain decomposition and
rank-to-device selection separately from mappings demonstrated at runtime.
Launch configuration alone will not be treated as evidence of correct physical
GPU assignment.
